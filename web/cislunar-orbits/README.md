# 地月空间轨道（栏目首页）

地月空间中的航天器不再近似为单一中心天体的开普勒轨道：地球与月球引力耦合使动力学呈现**限制性多体**特征，轨道族更加丰富，也对测控、燃料与任务节奏提出不同要求。本栏目在此建立共同语言，便于后续结合文献与工具深入具体设计方法。

## 与近地轨道的本质差异

- **引力模型**：从二体近似过渡到以地–月（及太阳）为主的多体模型；平动点、周期/拟周期轨道等概念变得核心。
- **时间与测控**：距离尺度大，信号时延与可见弧段分配成为常态约束。
- **稳定与机动**：部分轨道对初值敏感或需频繁维持；转移路径往往要在「时间、燃料、发射窗口」之间折中。

## 常见任务轨道族（概念层）

| 类型 | 含义（入门） | 延伸阅读 |
|------|----------------|----------|
| 地月转移 | 从近地或停泊轨道到月球影响球的过渡段 | 文献与任务报告中的「转移窗口」「中途修正」 |
| 绕月轨道 | 环月圆/椭圆、极地轨等，用于遥感、着陆准备 | 与月球引力场阶次、冻结轨道等相关 |
| 平动点与晕族轨道 | 相对地–月平动点附近的周期/拟周期轨道，可用于网关、中继与观测 | 参见 [NRHO](/glossary/nrho/) 等词条 |
| 远距离逆行轨道（DRO） | 相对地–月系逆行、尺度较大的族，适合演示与特定编队 | 参见 [DRO](/glossary/dro/) |

具体设计需结合星历、推力模型与任务条例；本站 [资源与工具](/resources-tools/) 汇总了常用数据与开源库，[卫星轨道仿真](/satellite-simulation/) 提供可交互的入门实验环境。


## 三大专题详解

### NRHO（近直线晕轨道）

NRHO（Near-Rectilinear Halo Orbit，近直线晕轨道）是一类在地月平动点附近运动的周期轨道，因其轨道形状在旋转坐标系中呈现细长的"近直线"特征而得名。这类轨道对月球南极的通讯可见性良好，且从轨道上前往月表的燃料消耗相对较低，因而成为美国"阿尔忒弥斯"计划中月球门户空间站的首选运营轨道。NRHO 本质上是限制性三体问题的解，轨道自身并非渐近稳定，需定期进行轨道维持，但其动力学特性已被大量任务验证，是当前地月空间运营的研究热点。

### DRO（远距离逆行轨道）

DRO（Distant Retrograde Orbit，远距离逆行轨道）是一类在旋转坐标系中逆行、距离地月质心较远的准周期轨道。与 NRHO 相比，DRO 距离月球较远，轨道维持需求较低，适宜作为地月空间的中长期"停车位"或深空任务的前哨站。由于其动力学结构相对宽松，DRO 在任务规划中常被用于演示多体轨道转移的可行性，也是研究地月空间混沌特性和轨道稳定性的重要对象。

### 地月转移轨道

地月转移轨道是航天器从近地停泊轨道进入月球影响球乃至绕月轨道的过渡通道。典型的转移方式包括霍曼转移、低能转移（月球转移轨道 LTO）以及近年来备受关注的多体引力辅助转移。不同转移方案在燃料消耗、时间成本和发射窗口灵活性上各有取舍：快速转移燃料消耗较大，低能转移虽然省燃料但耗时数月。实际任务需综合任务时序、运载能力与测控弧段选择最适方案。
