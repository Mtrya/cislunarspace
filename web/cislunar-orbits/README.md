# 地月空间轨道（栏目首页）

地月空间中的航天器不再近似为单一中心天体的开普勒轨道：地球与月球引力耦合使动力学呈现**限制性多体**特征，轨道族更加丰富，也对测控、燃料与任务节奏提出不同要求。本栏目在此建立共同语言，便于后续结合文献与工具深入具体设计方法。

## 与近地轨道的本质差异

- **引力模型**：从二体近似过渡到以地–月（及太阳）为主的多体模型；平动点、周期/拟周期轨道等概念变得核心。
- **时间与测控**：距离尺度大，信号时延与可见弧段分配成为常态约束。
- **稳定与机动**：部分轨道对初值敏感或需频繁维持；转移路径往往要在「时间、燃料、发射窗口」之间折中。

## 常见任务轨道族（概念层）

| 类型 | 含义（入门） | 延伸阅读 |
|------|----------------|----------|
| 地月转移 | 从近地或停泊轨道到月球影响球的过渡段 | 文献与任务报告中的「转移窗口」「中途修正」 |
| 绕月轨道 | 环月圆/椭圆、极地轨等，用于遥感、着陆准备 | 与月球引力场阶次、冻结轨道等相关 |
| 平动点与晕族轨道 | 相对地–月平动点附近的周期/拟周期轨道，可用于网关、中继与观测 | 参见 [NRHO](/glossary/nrho/) 等词条 |
| 远距离逆行轨道（DRO） | 相对地–月系逆行、尺度较大的族，适合演示与特定编队 | 参见 [DRO](/glossary/dro/) |

具体设计需结合星历、推力模型与任务条例；本站 [资源与工具](/resources-tools/) 汇总了常用数据与开源库，[卫星轨道仿真](/satellite-simulation/) 提供可交互的入门实验环境。
