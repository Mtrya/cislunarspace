## 公司概况

True Anomaly 是一家美国太空防御技术初创公司，成立于 2020 年左右，总部位于科罗拉多州。True Anomaly 是目前唯一一家专注于太空防御（space defense）领域的私营公司，其使命是为美国及其盟友提供在太空领域取得决定性优势所需的能力。公司于 2024 年 12 月完成了 1 亿美元的融资，但随即进行了约四分之一的裁员以精简运营。公司正在快速建设其太空飞行器和地面系统，目标是成为美国太空军的核心战术响应空间（ TacRS ）供应商之一。

## 太空与地月空间相关业务

### Jackal 自主轨道飞行器

True Anomaly 的核心产品是 Jackal 自主轨道飞行器：

- **定位**：Jackal 是一种用于太空领域态势感知（Space Domain Awareness，SDA）和轨道接近操作（Proximity Operations，PROX）的自主卫星平台。
- **任务能力**：Jackal 可自主接近目标航天器，执行近距离观测、成像和情报收集任务，支持美国太空军和情报机构的需求。2024 年第二次任务中，True Anomaly 成功部署了 Jackal 卫星并建立了通信，完成了关键里程碑。
- **蜂群协同**：True Anomaly 正在开发多卫星协同作业能力，支持多架 Jackal 同时执行复杂的轨道接近和感知任务。

### 战术响应空间（ TacRS ）

True Anomaly 正在积极参与美国太空军的战术响应空间（ Tactically Responsive Space， TacRS ）项目：

- **Victus Haze 任务**：2024 年 True Anomaly 宣布将支持军方的 Victus Haze 任务，该任务计划于 2025 年作为 TacRS 计划的一部分执行。萤火虫航空航天公司（Firefly Aerospace）的阿尔法火箭被选为 Jackal 飞行器的发射平台。
- **快速响应能力**：TacRS 的核心目标是展示从任务分配到发射在数天至数周内完成的快速响应能力，而无需数月的传统准备周期。

### 太空领域态势感知（SDA）

True Anomaly 通过其 Jackal 平台提供高分辨率的太空领域态势感知数据：

- **近距离观测**：通过轨道接近操作，Jackal 可对目标卫星进行高分辨率的光学成像，获取目标的具体构型和状态信息。
- **轨道行为分析**：对目标卫星的轨道机动行为进行长期跟踪和分析，识别异常活动模式。
- **地月空间潜力**：虽然 Jackal 目前主要在 LEO 地球近地轨道运行，其平台能力在概念上可扩展至更高的地月空间轨道。

## 主要产品与技术

| 产品/技术 | 类型 | 说明 |
|-----------|------|------|
| Jackal | 自主轨道飞行器 | 太空域态势感知和接近操作卫星 |
| GravityWorks | 地面运营中心 | 太空任务地面控制与数据分析 |
| 轨道机动算法 | 软件/算法 | 自主接近和轨道协调控制算法 |
| 多星协同系统 | 软件系统 | 多卫星协同感知和任务规划 |

## 与地月空间任务的关联

True Anomaly 在地月空间领域的关联主要体现在防御态势感知层面：

- **扩展跟踪能力**：其轨道接近和成像技术经过改进后可应用于地月空间的太空物体跟踪，填补目前的地月空间 SDA 能力空白。
- **轨道对抗能力**：True Anomaly 正在发展的轨道接近和机动能力，在对抗场景中可用于地月空间的轨道外科式精确打击（参考"轨道博弈"条目）。
- **快速响应架构**：TacRS 的快速响应理念对于地月空间的高价值资产保护具有重要意义，可以在数天内部署传感器或对抗载荷。
