# 中科宇航力箭二号超级工厂全面竣工，批量化生产线正式投产

**摘要：** 2026年4月28日，中科宇航位于绍兴市柯桥区的力箭二号大型液体运载火箭超级工厂全面竣工并正式投产。作为浙江省「千项万亿」工程重点项目，该工厂顺利完成研发、生产、总装、测试、运营等核心功能区建设，成熟运营后将具备年产12发力箭二号液体运载火箭的能力，标志着中国商业航天产业从技术验证迈向规模化生产的重要里程碑。就在一个多月前的3月30日，力箭二号遥一运载火箭已成功首飞，实现了「首飞告捷」与「产能落地」的高效衔接。

## 工厂概况与产能规划

力箭二号超级工厂位于浙江省绍兴市柯桥区经济技术开发区，是中科宇航在液体运载火箭产业化布局中的核心制造基地。工厂涵盖火箭贮箱、管路阀门、级间段、导管等液体火箭关键部件的生产区域，综合厂房内设置了火箭脉动式总装测试生产线，将火箭总装分解为多个固定工位，多枚火箭同步按节拍流转，实现并行、连续的产出。

力箭二号火箭副总师廉洁介绍，超级工厂投产后3至5年内，将持续提升力箭二号的生产效率，力争将总装测试周期缩短至2个月以内，实现满产12发的产能目标，支撑高频次、稳定的发射任务。

## 力箭二号火箭技术特点

力箭二号是我国首款采用「通用助推器核心」（CBC）构型的运载火箭，其技术参数如下：

| 参数 | 数值 |
|------|------|
| 通用芯级直径 | 3.35米 |
| 首飞状态整流罩直径 | 4.2米 |
| 总长 | 53米 |
| 起飞重量 | 625吨 |
| 起飞推力 | 753吨 |
| 500km太阳同步轨道运力 | 8吨 |
| 200km近地轨道运力 | 12吨 |

火箭采用通用化、组合化、模块化设计，可灵活拓展为0/2/4助推器捆绑模式，覆盖2至20吨的运力区间。通过模块共用，显著降低了研发成本。该火箭具备运载能力大、固有可靠性高、可制造性强、操作简洁便利、拓展空间大等优势。

## 首飞与量产的无缝衔接

就在一个多月前的2026年3月30日，力箭二号遥一运载火箭「国际纺都号」在东风商业航天创新试验区成功发射，将轻舟初样试飞船及2颗卫星精准送入预定轨道，发射任务取得圆满成功。首飞告捷后仅28天，超级工厂即实现竣工投产，创造了从「技术验证」到「产能落地」的中国商业航天新速度。

## 战略意义

该超级工厂的建成投用，不仅是中科宇航拓展业务版图、完善全产业链布局的关键一环，更是我国商业航天产业从技术验证迈向规模化生产的重要里程碑。随着批量化生产能力的形成，中国商业航天发射成本预计将从当前每公斤约2万美元下降至1万美元以下，大幅提升国际竞争力，为中国千帆星座等国家级重大工程提供有力支撑。
