# 我国成功发射卫星互联网技术试验卫星，手机宽带直连技术取得突破

**摘要：** 北京时间2026年4月24日14时35分，我国在西昌卫星发射中心使用长征二号D运载火箭，以一箭四星方式成功将卫星互联网技术试验卫星送入预定轨道，发射任务取得圆满成功。卫星主要用于开展手机宽带直连卫星、天地网络融合等技术试验验证，为我国卫星互联网建设与手机直连宽带服务奠定重要技术基础。

![长征二号D运载火箭成功发射卫星互联网技术试验卫星（澎湃新闻/许立豪 摄）](./figures/2026-04-24-satellite-internet-test-satellite/hero.jpg)
*图片来源：澎湃新闻 / 许立豪*

北京时间2026年4月24日14时35分，我国在西昌卫星发射中心使用长征二号D运载火箭点火起飞，随后以一箭四星方式将卫星互联网技术试验卫星顺利送入预定轨道，发射任务取得圆满成功。

## 任务目的与技术验证

此次发射的卫星互联网技术试验卫星，由银河航天（北京）科技集团股份有限公司承担研制，主要用于开展以下技术试验验证：

**手机宽带直连卫星**：验证普通智能手机直接连接卫星宽带网络的技术可行性。未来用户在偏远地区、海洋、航空等无法接入地面基站的场景下，有望直接通过手机连接卫星获得高速宽带接入服务，无需额外设备。

**天地网络融合**：构建地面通信网络与卫星通信网络的一体化架构，实现用户在两种网络之间的无缝切换，是未来6G天地一体化网络的关键技术储备。

此次任务的成功实施，标志着我国在手机宽带直连卫星技术领域取得重要突破，为后续卫星互联网建设和商用服务奠定了坚实基础。

## 任务基本信息

| 项目 | 详情 |
|------|------|
| 运载火箭 | 长征二号D（LM-2D） |
| 发射场 | 西昌卫星发射中心 |
| 发射时间 | 2026年4月24日 14:35 BJT |
| 载荷配置 | 一箭四星 |
| 卫星用途 | 手机宽带直连、天地网络融合技术试验 |
| 研制单位 | 银河航天（北京）科技集团 |
| 任务意义 | 卫星互联网建设重要技术验证 |

长征二号D运载火箭由中国航天科技集团有限公司八院抓总研制，是一型常温两级液体运载火箭，具备适应不同轨道要求的单星及多星发射能力，500公里太阳同步轨道运载能力为1.9吨。本次发射是长征二号D运载火箭的第104次发射，也是长征系列运载火箭的第639次飞行。
