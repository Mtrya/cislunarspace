# 力箭二号遥一运载火箭首飞成功，一箭三星送入预定轨道

**摘要：** 2026年3月30日19时00分，中科宇航自主研制的力箭二号遥一运载火箭在位于内蒙古阿拉善盟额济纳旗的东风商业航天创新试验区点火发射，将新征程01卫星、新征程02卫星和天视卫星01星共3颗卫星精准送入预定轨道，飞行试验任务取得圆满成功。力箭二号是中国首款采用"通用助推器核心"（CBC）构型的运载火箭，首飞即服务于国家重大战略和重大工程建设。

![Artemis II 火箭转运至发射工位（NASA / Kennedy Space Center）](./figures/2026-03-30-lijian-2-maiden-flight/hero.jpg)
*图片来源：NASA / Kennedy Space Center（公共领域图片，用于说明全球火箭发射场景）*

## 任务概况

力箭二号（PR-2）遥一运载火箭于北京时间2026年3月30日19时00分在东风商业航天创新试验区发射升空，约数分钟后，火箭将搭载的新征程01卫星、新征程02卫星和天视卫星01星精准送入预定轨道，发射任务取得圆满成功。

此次任务是力箭二号运载火箭的首次飞行，也是力箭系列运载火箭第12次发射。

## 火箭参数

力箭二号遥一运载火箭首飞状态配置：

| 参数 | 数值 |
|------|------|
| 整流罩直径 | 4.2米 |
| 芯级直径 | 3.35米 |
| 一子级带助推器直径 | 5.8米 |
| 总长 | 53米 |
| 起飞质量 | 625吨 |
| 级数 | 两级半 |
| 近地轨道载荷 | 12吨 |
| 太阳同步轨道载荷 | 8吨（500km）/ 5.6吨（700km） |

## 技术亮点

力箭二号作为中国首款"通用助推器核心"（Common Booster Core，CBC）构型的运载火箭，其通用芯级直径3.35米，具备模块化、通用化特点，可通过助推器数量调整实现不同运力配置。

力箭二号副总师廉洁表示，此次首飞即服务于国家重大战略和重大工程建设，火箭在设计上实现了大运力与低成本的平衡，将探索新型天地运输模式。

## 商业意义

力箭二号的首飞成功标志着中国民营商业火箭从"试验品"向"实用型"加速转变。随着可回收火箭技术进入密集验证期、商业航天供应链逐步成熟，中国商业航天正在从技术验证阶段迈入规模化工程阶段，为2026年全年商业发射超60次的目标提供有力支撑。
