# 长征二号D一箭四星，成功发射卫星互联网技术试验卫星

**摘要：** 北京时间2026年4月24日14时35分，我国在西昌卫星发射中心使用长征二号D运载火箭，以一箭四星方式成功将卫星互联网技术试验卫星送入预定轨道，发射任务取得圆满成功。卫星主要用于开展手机宽带直连卫星、天地网络融合等技术试验验证。此次任务是长征系列运载火箭的第639次飞行。

![长征二号D运载火箭发射升空（澎湃新闻/许立豪 摄）](./figures/2026-04-24-long-march-2d-launch/hero.jpg)
*图片来源：澎湃新闻 / 许立豪*

北京时间2026年4月24日14时35分，西昌卫星发射中心，长征二号D运载火箭点火起飞，随后以一箭四星方式将卫星互联网技术试验卫星顺利送入预定轨道，发射任务取得圆满成功。

此次发射的卫星由银河航天（北京）科技集团股份有限公司承担研制，主要用于开展手机宽带直连卫星、天地网络融合等技术试验验证。这是我国卫星互联网建设的重要技术验证环节，为后续手机直连卫星宽带服务奠定基础。

## 技术意义

卫星互联网技术试验卫星的成功发射，标志着我国在手机宽带直连卫星技术领域取得重要突破。手机宽带直连卫星技术将使普通智能手机直接连接卫星宽带网络成为可能，用户在偏远地区、海洋、航空等场景下也能获得高速宽带接入服务，无需依赖地面基站。

天地网络融合技术则致力于构建地面通信网络与卫星通信网络的一体化架构，实现用户在两者之间的无缝切换，是未来6G天地一体化网络的重要组成部分。

## 任务概况

| 项目 | 信息 |
|------|------|
| 运载火箭 | 长征二号D（LM-2D） |
| 发射场 | 西昌卫星发射中心 |
| 发射时间 | 2026年4月24日 14:35 BJT |
| 载荷 | 卫星互联网技术试验卫星 ×4 |
| 轨道 | 预定轨道 |
| 研制单位 | 银河航天等 |
| 任务编号 | 长征系列火箭第639次飞行 |

长征二号D运载火箭由中国航天科技集团有限公司八院抓总研制，是一型常温两级液体运载火箭，具备适应不同轨道要求的单星及多星发射能力，500公里太阳同步轨道运载能力为1.9吨。
