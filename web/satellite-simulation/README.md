本页提供**交互式轨道仿真**教学组件，用于建立对轨道要素、轨迹形状与可视化尺度的直觉。它侧重教学演示，**不替代**专业任务分析软件（如完整力模型、测控弧段与燃料优化等需在专用环境中完成）。
