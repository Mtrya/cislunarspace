# 轨道设计与优化

## 研究背景

轨道设计是地月空间任务的核心技术之一，涵盖从近地轨道到月球轨道的各类转移、驻留和返回轨道的设计与优化。

## 研究内容

- 低能转移轨道设计
- 轨道参数表征方法
- DRO/NRHO 等特定轨道族设计
- 轨道优化算法
